## vue-cli脚手架

### 认识cli

包含常用的vue工程模板，可以快速启动一个项目



### 使用vue-cli

安装

&#124;cnpm i vue-cli -g[^1] 

列出模板

&#124;vue list

| 名称                         | 说明                               |
| :--------------------------- | :--------------------------------- |
| browserify/browserify-simple | vue+browserify模板，提供热更新服务 |
| pwa                          | vue+pwa模板                        |
| simple                       | 空白模板                           |
| webpack/webpack-simple       | vue+webpack，提供一切需要的功能    |

创建项目

&#124;vue init <模板> <目录>




























[^1]: 全局安装，通过&#124;npm config get prefix 命令可以查看全局安装位置
